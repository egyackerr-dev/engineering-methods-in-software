# Practical Examples for One-Shot Prompts

Below are ready-to-use realistic examples using the one-shot-prompts.md template.

---

## Example 1:
Generate a single portable Python script named `data_cleaner_v2.0.py` to:
Clean a CSV file by removing rows with missing or invalid data values.

---

## Example 2:
Generate a single portable Bash script named `backup_checker.sh` to:
Backup files from a source to a destination folder and verify backup success, handling all possible errors.

---

## Example 3:
Generate a single portable JavaScript file named `image_resizer.js` to:
Resize images in a directory to a specified resolution, validating inputs and reporting processing status to the user.

---

## Example 4:
Generate a single portable TypeScript file named `api_monitor.ts` to:
Monitor an external API endpoint, alert on failures or delays, and save logs, ensuring total reliability and detailed reporting.

---

## Example 5:
Generate a single portable Python script named `user_authenticator.py` to:
Authenticate users by secure password input, handle all errors and edge cases, provide friendly prompts and full logging.
